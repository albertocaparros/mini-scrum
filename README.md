# Mini-Scrum

A modern Angular v20 application for managing Scrum projects, demonstrating advanced development practices.

## Features

- Product Backlog management
- Sprint Backlog selection
- State management with Signals
- Testing with Vitest
- Linting with ESLint and formatting with Prettier

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/mini-scrum.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
